
Degee_project_ML содержит файлы дипломного проекта Netology по анализу Global Innovation index всех стран с 2013 по 2019 г.
Ноутбук с проектом лежит в корне папки.

Ссылка на сайт GII: https://www.globalinnovationindex.org
Ccылка на датасеты для загрузки: https://www.globalinnovationindex.org/analysis-indicator

Структура каталога:
data\full - полные датасеты с данными по всем странам
data\short - сокращённые дататсеты, содержащие:
   rank	- ранг страны (по индеку GII)
   Economy - название страны
   Income Group(Strength/Weakness) - рост рейтинга (Strength) или ослабление (Weakness) по сравнению с предыдущем годом
   Value - ?
   Score - показатель GII

Задача 1:
Предлагаю преобразовать данные за полученные года по каждым странам. То есть, возьми score по каждому подпункту GII, таким образом ты создашь признаки, которые содержат оценки по этим номинациям. Также добавь признаки год исследования, и страна. Первое, что надо будет сделать после этого, попробуй обучить элементарный kMeans, убрав признаки год и страну, а также выбирая число кластеров, и посмотреть, что получится. Скорей всего, какие-то страны в разные года могут оказаться в разных кластерах, это нормально.
