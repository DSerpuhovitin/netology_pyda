
Degee_project_ML содержит файлы дипломного проекта Netology по анализу Global Innovation index всех стран с 2013 по 2019 г.
Ноутбук с проектом в корне папки.

Задача:  Найти характерные показатели государственной поддержки подъиндексов GII при пазных спосоах кластеризации. 

Ссылка на сайт GII: https://www.globalinnovationindex.org
Ccылка на датасеты для загрузки: https://www.globalinnovationindex.org/analysis-indicator

Структура каталога:
data\full - полные датасеты с данными по всем странам
data\short - сокращённые дататсеты, содержащие:
   rank	- ранг страны (по индеку GII)
   Economy - название страны
   Income Group(Strength/Weakness) - рост рейтинга (Strength) или ослабление (Weakness) по сравнению с предыдущем годом
   Score - показатель GII

Цель работы: выявление наиболее значимых показателей государственной поддекржки национальной инновационной системы.
